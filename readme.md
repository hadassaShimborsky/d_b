## Installation

To get started with the project, clone the repository and install the dependencies.
```bash
git clone https://github.com/yourusername/your-repo.git
cd d_b
```

install modules
```bash
cd client
npm install
cd ../server
npm install
cd ../
npm install
```

run client and server:

```bash
npm start
```

client is running on http://localhost:3000/
